# IT Labor Market in Armenia (2022–2025)

An interactive data story exploring how Armenia’s IT labor market evolved between 2022 and 2025 under the influence of migration waves, employment growth, salary dynamics, and overall market maturation.

The project highlights key trends shaping the emergence of a more sustainable and resilient tech ecosystem in Armenia.

## Project Scope
- IT workforce growth and structure  
- Salary dynamics across the sector  
- Impact of migration waves  
- Market maturation and sector comparison  

## Data Source
All statistical data used in this project is based on publicly available information from the **Statistical Committee of the Republic of Armenia**.

## Technology
- React  
- TypeScript  
- Vite
